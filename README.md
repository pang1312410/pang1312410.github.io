# 🦖 Science Land - สื่อการเรียนรู้วิทยาศาสตร์ ป.2

เว็บไซต์สื่อการเรียนการสอนวิทยาศาสตร์ระดับประถมศึกษาปีที่ 2 ในรูปแบบ Interactive Web Application ที่ออกแบบด้วยโทนสีพาสเทลสดใส น่ารัก เหมาะสำหรับเด็กปฐมวัย/ประถมศึกษาตอนต้น เพื่อส่งเสริมทักษะและทบทวนความรู้เกี่ยวกับ:
1. **การเรียนรู้สิ่งต่าง ๆ รอบตัว** (ทักษะกระบวนการทางวิทยาศาสตร์ และการสืบเสาะหาความรู้)
2. **วัสดุและการใช้ประโยชน์** (สมบัติการดูดซับน้ำ, สมบัติจากการผสมวัสดุ, การเลือกใช้วัสดุ)

```text
├── README.md               # เอกสารประกอบโครงการ
├── assets/
│   ├── css/
│   │   └── styles.css      # สไตล์ทั้งหมดของเว็บ (Pastel Theme, Games, responsive)
│   ├── js/
│   │   ├── app.js          # จัดการระบบเปลี่ยนหน้า (Routing), คะแนน, ใบเกียรติบัตร
│   │   ├── content.js      # ข้อมูลเนื้อหาและชุดคำถามแบบทดสอบ (ภาษาไทย)

│   └── images/

```

## 💻 วิธีการเปิดใช้งานและทดสอบในเครื่องคอมพิวเตอร์ (Local Testing)
เนื่องจากตัวเว็บไซต์ใช้ระบบ JavaScript ES6 Modules (`import`/`export`) เบราว์เซอร์ส่วนใหญ่จะไม่อนุญาตให้เปิดใช้งานตรงๆ ผ่านการดับเบิ้ลคลิกไฟล์ `index.html` (เนื่องจากติดปัญหา CORS Policy ของเบราว์เซอร์เมื่อรันบนโปรโตคอล `file://`) 

คุณสามารถเปิดใช้งานได้อย่างง่ายดายด้วยสคริปต์เซิร์ฟเวอร์น้ำหนักเบาที่เราจัดเตรียมไว้ให้:
1. เปิดโปรแกรม Command Prompt, PowerShell หรือ Terminal ในโฟลเดอร์นี้
2. พิมพ์คำสั่งรันเซิร์ฟเวอร์:
   ```bash
   node serve.js
   ```
3. เปิดเว็บเบราว์เซอร์แล้วเข้าไปที่ลิงก์:
   ```text
   http://localhost:3000/
   ```

## 🚀 วิธีการอัปโหลดและ Deploy ไปยัง GitHub Pages
ระบบ Git และคอมมิตแรกได้รับการเตรียมความพร้อมภายในโฟลเดอร์นี้เรียบร้อยแล้ว หากต้องการผลักข้อมูล (Push) ขึ้นไปยัง Repository ของคุณเพื่อเปิดใช้งาน GitHub Pages ให้ดำเนินการตามขั้นตอนด้านล่างนี้:

1. เปิด Terminal หรือ Git Bash ในโฟลเดอร์ของโครงการนี้
2. ใช้คำสั่งเพื่อ Push ไฟล์ไปยัง GitHub (อาจจำเป็นต้องเข้าสู่ระบบบัญชี GitHub ของคุณหากยังไม่ได้ตั้งค่า):
   ```bash
   git push -u origin main
   ```
3. เมื่ออัปโหลดสำเร็จแล้ว ให้เข้าไปที่ GitHub Repository ของคุณในเว็บบราวเซอร์:
   `https://github.com/pang1312410/pang1312410.github.io`
4. ไปที่เมนู **Settings** (การตั้งค่า) ของ Repository
5. เลือกเมนู **Pages** ในแถบด้านซ้าย
6. ในส่วนของ **Build and deployment**:
   - Source: เลือก `Deploy from a branch`
   - Branch: เลือกสาขา `main` และโฟลเดอร์เป็น `/ (root)`
   - กดปุ่ม **Save**
7. รอระบบประมวลผลประมาณ 1-2 นาที เว็บไซต์จะแสดงผลให้ใช้งานได้ทันทีผ่านลิงก์ของคุณ:
   👉 **`https://pang1312410.github.io/`**
